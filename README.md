# Data Analytics Project 1
This project was completed as part of the DecodeLabs Data Analytics Internship.
The objective was to clean and prepare a raw dataset by identifying missing values, removing duplicate records, and correcting inconsistent data formats using Python and Pandas.

 Technologies Used:
- Python
- Pandas
- Git
- GitHub

 Project Files

- `Dataset for Data Analytics - Sheet1.csv` – Original dataset
- `Cleaned_Dataset.csv` – Cleaned dataset after preprocessing
- `project1.py` – Python script used for data cleaning

 Data Cleaning Tasks
The following tasks were completed:
- Identified and handled missing values
- Removed duplicate records
- Corrected data formats where needed
- Saved the cleaned dataset for further analysis

How to Run:
1. Install the required library:
pip install pandas

2. Run the script:
python project1.py

Project Outcome:
The final dataset is cleaned and ready for analysis. The project demonstrates the basic data preparation techniques commonly used before performing data analysis.
